# ApplicationPortal

**Features**
  * Students can apply for leave and check the current status.
  * Faculty can approve or deny requests.

## Requirements
    certifi==2018.10.15
    chardet==3.0.4
    Django==3.0.8
    django-crispy-forms==1.7.2
    idna==2.7
    Pillow==5.2.0
    pytz==2018.5
    requests==2.19.1
    urllib3==1.23
    
## Run pip install -r requirements.txt to install necessary requirements
